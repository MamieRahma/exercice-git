Présentation du projet : Module Python de tri et recherche
1️⃣ Introduction
Objectif : créer un module Python capable de trier et rechercher des éléments dans une table à deux dimensions.
Utilité : faciliter l’organisation et la recherche de données dans des tableaux, ce qui est courant en informatique et en sciences de données.
2️⃣ Les algorithmes de tri implémentés
Tri à bulle (Bubble Sort)
Compare les éléments deux à deux et les échange si nécessaire.
Simple mais lent pour de grandes tables.
Tri rapide (Quick Sort)
Divise le tableau en sous-tableaux selon un pivot.
Très efficace pour les grandes tables.
Tri par sélection (Selection Sort)
Cherche l’élément le plus petit et le place à la bonne position.
Facile à comprendre mais moins rapide que Quick Sort.
Tri par insertion (Insertion Sort)
Insère chaque élément à sa place dans une liste triée.
Efficace pour des petits tableaux presque triés.
Tri par fusion (Merge Sort)
Divise le tableau en deux moitiés, trie chaque moitié, puis fusionne.
Rapide et stable pour toutes les tailles.
(💡 Pour chaque algorithme, j’ai mis un commentaire dans le code expliquant le fonctionnement avec un exemple de tableau 2D.)
3️⃣ Les algorithmes de recherche
Recherche linéaire
Vérifie chaque élément un par un.
Simple mais lent pour de grands tableaux.
Recherche binaire
Nécessite un tableau trié.
Divise le tableau en deux pour trouver l’élément.
Très rapide pour des grandes listes triées.
Recherche par dichotomie
Même principe que la recherche binaire, avec un nom différent selon le contexte.
(💡 Les commentaires expliquent aussi comment rechercher dans une table à deux dimensions.)

5️⃣ Conclusion
Module Python complet pour trier et chercher facilement dans des tableaux 2D.
Facile à importer et réutiliser dans n’importe quel projet Python.
Permet de comprendre les bases des algorithmes de tri et recherche.
