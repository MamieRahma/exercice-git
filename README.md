# Projet Laravel

## Description
Ce projet consiste à initialiser une application web avec Laravel (back-end) et un front-end intégré.

## Objectifs
- Mise en place d’un projet Laravel
- Gestion du front-end avec Vite
- Versionnement avec Git
- Hébergement sur GitHub

## Outils utilisés
- Laravel
- PHP
- Git
- GitHub
- Node.js / npm
